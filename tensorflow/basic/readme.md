### Eager Execution VS Graph Execution
| 특징               | Eager Execution                                  | Graph Execution                                 |
|--------------------|-------------------------------------------------|------------------------------------------------|
| **실행 방식**       | 연산이 즉시 실행                                  | 그래프를 빌드한 후 한 번에 실행                    |
| **디버깅**          | 직관적이며 쉽다                                    | 복잡하며 어려울 수 있음                            |
| **성능**           | 최적화되지 않음                                    | 그래프 최적화로 높은 성능                          |
| **코드 복잡도**     | 간단하고 직관적                                    | 복잡할 수 있음                                    |
| **배포**            | 배포에 덜 유리함                                    | 배포에 유리함                                     |
